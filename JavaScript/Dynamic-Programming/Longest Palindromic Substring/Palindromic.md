# 5. Longest Palindromic Substring

- Palindromic Substring - When it reads the same read backwards or forward.

Question:
    - Given a string s, return the longest palindromic substring in s.

Constraints:
    - 1 <= s.length <= 1000

Examples:
    Input: s = "babad"
    Output: "bab"/"aba"

    Input: "cbbd"
    Output: "bb"
